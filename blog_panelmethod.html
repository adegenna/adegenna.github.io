Bayesian optimization for (low-speed) aerodynamics

In this post, I want to walk through a classic problem in aerodynamics -- wing shape optimization -- through the lens of a modern technique in robust optimization called Bayesian optimization. Looking back, one of the things I wished I'd had to play with while I was in undergrad was a "Goldilocks" computer model: one which is complex enough to be interesting, but still simple enough you can actually do it with modest effort. I'm hoping I get that balance right, so that this can be useful to anyone who wants an accessible entry point into the intersection of aerodynamics, uncertainty quantification, and optimization. If you'd like to experiment for yourself, the code is [here].
I'll describe the problem first, then talk about the physics solver we'll use to model the aerodynamics (a panel method), then talk about the method we'll use for shape optimization (Bayesian optimization), and then we'll put all the pieces together.
